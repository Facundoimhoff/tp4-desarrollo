Backend:

API RESTful desarrollada con Node.js, Express y Sequelize para la gestión de un torneo de fútbol, conectada a una base de datos local SQLite.

Instalación y Ejecución:
1. Abrir una terminal en la carpeta del backend "TP4-DESARROLLO/torneo-backend"
2. Instalar las dependencias del proyecto:
   bash
   npm install
3. Escribir en la terminal "node App.js" , luego saltará un mensaje el cual dice "Servidor escuchando en ..." PUERTO 3000!!!!!!

Frontend:

Interfaz de usuario desarrollada con React y React Bootstrap para consumir la API de nuestro torneo local.

Instalación y Ejecución

1. Abrir una terminal en la carpeta del backend "TP4-DESARROLLO/torneo-frontend"
2. Instalar las dependencias del proyecto:
   bash
   npm install
3. Escribir en la terminal "npm install" y luego "npm run dev", luego presionar las teclas O + Enter.
